# LadderLad Library

This library provides a ladder logic simulation engine based on `ladder_logic.py`. It supports compiling ladder schematics to an instruction list and simulating the logic execution.

## Components

- `ladderlad/parser.py`: Parses ASCII ladder schematics.
- `ladderlad/renderer.py`: Renders instructions back to ASCII schematics.
- `ladderlad/engine.py`: Orchestrates parsing and rendering.
- `ladderlad/simulator.py`: Simulates the execution of the compiled instruction list.
